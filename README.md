# Curso Práctico de PHP y JavaScript, marzo-abril de 2026 🚀
Repositorio del curso de programación web con PHP y JavaScript de 2026

Este repositorio contiene el código evolutivo del curso. Aquí encontrarás los ejemplos, ejercicios y la aplicación que estamos construyendo paso a paso.

## 🛠️ Requisitos previos
Para ejecutar este proyecto localmente, asegúrate de tener instalado:
* **XAMPP** (Última versión disponible con PHP 8.x).
* **Visual Studio Code**.

## 💻 Configuración del entorno
Para seguir el curso con la misma experiencia de desarrollo, recomendamos instalar las siguientes extensiones en VS Code:
1. **PHP Intelephense**: Para el autocompletado y análisis de código.
2. **Error Lens**: ¡Fundamental! Para ver los errores de sintaxis directamente en la línea de código.
3. **Auto Rename Tag**: Jung Han. Muy útil para ahorrar trabajo editando etiquetas HTML.


## 📚 Contenido del Curso
El enfoque principal de este curso es la lógica de programación y la integración entre el backend y el frontend:
* **Backend:** PHP (Estructuras, funciones, manejo de datos).
* **Frontend:** JavaScript (DOM, eventos, Fetch API).
* *Nota: Usamos HTML y CSS de forma auxiliar para dar estructura, pero no son el foco principal del estudio.*

## 🚀 Cómo empezar
1. Clona este repositorio dentro de tu carpeta `C:/xampp/htdocs/`.
2. Inicia los módulos **Apache** y **MySQL** desde el Panel de Control de XAMPP.
3. Abre el navegador y accede a `http://localhost/nombre-de-tu-carpeta`.
